IIFE Intro

Immediately Invoked Function Expression

  • a JavaScript function that runs as soon as it is defined
  • a function that is written as an expression that runs right away
  • instead of writing a function declaration or function expression and then calling later
  • IIFE = you don't invoke/call it separately
  • creates new scope and creates “privacy”

IIFE Augmentation Pattern

To Augment One Single Variable on the Global Scope

  • Define IIFE in the first (main) file. Return a literal object.
  • In 2nd file, pass original function name in at bottom.
  • Original function name becomes argument at the top of new function. (Give it a slightly different name.)
  • Augment it with new key: value pairs.
  • Return it right back out again.

Makes it easier for mutliple developers on a team to work on functionality and add/modify features by splitting up into multiple JS files where each JS file deals with only one specific feature.
